npm install错误之node版本问题

npm install时报错

出错原因

node版本过高导致，可以通过node -v查看版本

解决方法

重新安装node.js

1.找到node.js，卸载

2.到官网重新下载版本更低的node.js

官网地址

新的错误

这时候重新运行npm install，又出现新的报错

解决方法

1.删除项目文件的node_modules
2.删除package-lock.json文件
3.清除npm缓存，执行npm cache clean –force
4.再运行npm install

成功
